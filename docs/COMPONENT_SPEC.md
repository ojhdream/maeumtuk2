# COMPONENT_SPEC.md

> 마음툭 UI 컴포넌트 표준

## 1. App Bar
- 브랜드: 마음(네이비) + 툭(초록) + 노란 점
- 높이: 56px
- 검색은 기본 노출하지 않고 돋보기 버튼으로 진입

## 2. FAB
역할: 가장 빠른 기록 진입

액션
- 새 툭 남기기
- 최근 툭 이어쓰기
- 이어툭 남기기

항상 우측 하단 고정.

## 3. Bottom Navigation
탭
- 오늘
- 툭로그
- 요즘
- 나

높이 78px
라운드 30px

## 4. Compose
필수 요소
- 큰 입력창
- 날짜 변경
- 사진
- 손으로
- 마음 이름(선택)

키보드가 올라와도 도구 영역은 사용 가능해야 한다.

## 5. Filter Chip
- 한 줄 유지
- 가로 스크롤
- 줄바꿈 금지
- 선택 상태는 노란 점과 부드러운 배경 강조

## 6. Tuk Card
포함
- 날짜
- 내용
- 타입(글·사진·손으로)
- 이어툭
- 더보기(수정/삭제)

## 7. 모인 마음 카드
역할: 들어가서 보는 공간
표시
- 제목
- 툭 개수
- 사진 개수
- 이어툭 개수
- 최근 기록일

## 8. 요즘 머문 말
역할: 이번 주 흐름
한 문장으로 자연스럽게 보여준다.
단어 목록만 나열하지 않는다.

## 9. AI 카드
절대 분석처럼 보이면 안 된다.
"발견" 톤 유지.

## 10. Empty State
비난하지 않는다.
예)
- 한 단어여도 괜찮아요.
- 오늘은 여기 두고 가세요.

## Component Rules
- 카드보다 여백을 우선한다.
- 그림자는 약하게.
- 애니메이션은 짧고 조용하게.
- 브랜드 컬러는 절제해서 사용한다.
